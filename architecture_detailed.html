SSO & Security - Keycloak + Istio

SSO Login Flow - Nguoi dung login 1 lan, truy cap tat ca dashboards

Buoc 1: User truy cap bat ky dashboard

1 User go: grafana.face-detect.dev
2 Istio EnvoyFilter bat request -> check cookie _oauth2_proxy
3 Khong co cookie -> redirect den auth.face-detect.dev/realms/face-detection/protocol/openid-connect/auth

Buoc 2: Keycloak xac thuc

4 User nhap username/password (hoac SSO qua Google/GitHub/LDAP)
5 Keycloak xac thuc -> tao JWT token chua: user_id, roles[], groups[]
6 Redirect ve grafana.face-detect.dev voi authorization code

Buoc 3: OAuth2 Proxy xu ly token

7 OAuth2 Proxy (sidecar trong Istio) doi code lay access_token + id_token
8 Set cookie _oauth2_proxy cho domain *.face-detect.dev
9 Cac lan sau: cookie hop le -> khong can login lai -> SSO across ALL dashboards

Buoc 4: RBAC - Phan quyen theo Role

10 JWT chua role claim: {"roles": ["ds-role"]}
11 Moi dashboard doc role tu JWT -> hien thi features tuong ung
12 VD: ds-role thay MLflow + Kubeflow, nhung KHONG thay Jenkins admin

So sanh: Cach hien tai vs. Dung deployKF

Cach hien tai: Config thu cong tung tool

- Helm install Kubeflow rieng
- Helm install MLflow rieng
- Helm install Airflow rieng
- Helm install Istio rieng
- Config Keycloak OIDC cho tung tool
- Tu viet Terraform/Ansible cho tung component
- Tu manage version compatibility
- Uu diem: Hieu sau, control tung chi tiet, tot cho hoc tap
- Nhuoc diem: Mat thoi gian, de gap version conflict

Dung deployKF: 1 command deploy tat ca

- 1 values.yaml chua toan bo config
- ArgoCD tu dong sync + deploy
- Dex SSO da tich hop san (thay Keycloak)
- Istio da config san
- MinIO da tich hop san
- Upgrade in-place: doi version trong values.yaml
- Uu diem: Nhanh, it loi, auto-upgrade, production-ready
- Nhuoc diem: Phu thuoc ArgoCD, kho custom deep, Dex thay vi Keycloak

Khuyen nghi cho do an: Voi do an hoc tap, nen lam CA HAI cach: (1) Dung deployKF de co baseline nhanh va hieu cach mot production platform hoat dong, (2) Tu config tung tool de hieu sau. Trong bao cao, so sanh 2 cach se la diem cong lon.
